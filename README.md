# NBA Stats React Web Application

## Flask

Need to activate virtual env each time you run project.
To start: source venv/bin/activate
To stop: deactivate


To run: 
``` bash
python flask-server/server.py
```


## React

To start: 
``` bash
cd client
npm start
```


### If 'Address already in use' error

``` bash
npx kill-port 5000
npx kill-port 3000
```
